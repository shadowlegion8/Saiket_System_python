# Task 5 - Currency Converter

## 📌 Overview

This project was completed as part of my internship at **Saiket Systems**. It is a Python-based Currency Converter that converts one currency into another using real-time exchange rates fetched from an online API.

The project allows users to enter source currency, target currency, and amount, then displays the converted result accurately.

---

## 🚀 Features

* Real-time currency conversion
* Supports multiple international currencies
* User-friendly input system
* Accurate exchange rate calculation
* Handles invalid currency codes
* Error handling for network/API issues

---

## 🛠️ Technologies Used

* Python
* Requests Library
* Exchange Rate API
* JSON Data Handling
