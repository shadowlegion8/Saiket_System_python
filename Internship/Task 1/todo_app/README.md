📝 To-Do List Application
📌 Description

This is a simple To-Do List application built using Python.
It allows users to add tasks, view tasks, and mark them as completed through a menu-driven interface.

🚀 Features
Add new tasks
View all tasks
Mark tasks as completed
Delete tasks
Edit existing tasks
Persistent storage using JSON file
🛠 Technologies Used
Python
JSON (for data storage)
📂 Project Structure

todo_app/
│── main.py
│── tasks.json
│── README.md

▶️ How to Run the Project
Open terminal or command prompt
Navigate to project folder
Run the program:
python main.py
💡 How It Works
Tasks are stored using a Python class
Data is saved in a tasks.json file
The application provides a menu to perform operations
