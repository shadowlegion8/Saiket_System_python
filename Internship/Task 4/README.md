# Task 4 - Basic Web Scraper

## 📌 Overview

This project was completed as part of my internship at **Saiket Systems**. It is a Python-based web scraper that extracts useful information from websites such as page titles, headings, and links by entering any valid URL.

The scraper uses **Requests** and **BeautifulSoup** libraries to fetch and parse HTML content efficiently.

---

## 🚀 Features

* Accepts any website URL as input
* Extracts page title
* Scrapes headings (H1, H2, H3)
* Extracts website links
* Handles HTTP errors including 403 issues
* Saves scraped data into CSV file

---

## 🛠️ Technologies Used

* Python
* Requests
* BeautifulSoup4
* CSV Module
