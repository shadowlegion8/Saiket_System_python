# Task 6 - Word Count Tool

## 📌 Overview

This project was completed as part of my internship at **Saiket Systems**. It is a Python-based Word Count Tool that analyzes a text file and provides useful statistics such as total lines, total words, total characters, and most frequently used words.

The project helps in understanding text analysis, file handling, and data processing using Python.

---

## 🚀 Features

* Reads text file provided by user
* Counts total lines
* Counts total words
* Counts total characters
* Displays top 10 most common words
* Handles invalid file name errors
* Structured output format

---

## 🛠️ Technologies Used

* Python
* Collections Module (Counter)
* File Handling
* String Manipulation
